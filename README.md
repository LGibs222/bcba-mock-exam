# BCBA Mock Exam — 6th Edition Test Content Outline

Interactive 522-question practice exam for BCBA certification, aligned with the 6th Edition Test Content Outline.

## Features

- **Timed Exam Mode** — 4-hour countdown simulating real testing conditions
- **Study Mode** — Untimed practice with instant feedback after each answer
- **Progress Saving** — Auto-saves to localStorage; resume where you left off
- **Question Randomization** — Shuffled each attempt for varied practice
- **Flag & Review** — Bookmark questions and revisit before submitting
- **Detailed Results** — Domain-by-domain breakdown with performance bars
- **Results Export** — Download CSV summary of your performance
- **Keyboard Shortcuts** — A/B/C/D to answer, arrows to navigate, F to flag, M for map
- **Mobile Friendly** — Swipe navigation, responsive layout, bottom-sheet question map
- **Dark Mode** — Automatic via system preference
- **Accessible** — ARIA roles, keyboard-navigable options
